About Me

Nama saya Fira Mardiani. Saya lahir di Sengkubang, 29 Maret 2001 . Saya tinggal di Jl. Dr Rubini Gang Sepakat No. 40, Mempawah, Kalimantan Barat. Jenis kelamin saya Perempuan. Saya beragama Islam.

Pekerjaan saya saat ini adalah Sebagai Mahasiswa di Sekolah Tinggi Ilmu Ekonomi Indonesia Pontianak ( STIE Indonesia Pontianak ) dengan program studi S1 Manajemen.

Saya anak pertama dari tiga bersaudara. Ayah saya bernama Tomi dan ibu saya bernama Arningsih.

Riwayat pendidikan :

- SD Negeri 21 Mempawah Hilir ( Dari Tahun 2007 - Tahun 2013 )

- SMP Negeri 2 Mempawah Hilir ( Dari Tahun 2013 - Tahun 2016 )

- SMA Negeri 2 Mempawah Hilir ( Dari Tahun 2016 - 2019 )

- Sekolah Tinggi Ilmu Ekonomi Indonesia Pontianak ( Dari Tahun 2019 - Sekarang )
